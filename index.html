Projects

JeePS: Realtime Public Transportation Tracking System

September 2023 - June 2024

Created a web application and mobile application that improves the overall situation of public informal transportation in the Philippines. The system addresses problems of passengers regarding safety, driver’s behavior, information materials and service adequacy by providing several information using Flutter and Firebase. In order to adress these factors, several features were developed. These features include: Realtime GPS Location, Fare Matrix, Basic Information, Passenger Counter, Rough Time Estimation, Feedback System (5 star system and text feedback), Report System, Passenger Demand System, Account System, and various Route Manager Features like editing the route and verifying drivers.

MeralCOST: Analysis on Meralco Typical Consumption Price

March 2024 - June 2024

Used Data Science to analyze and interpret data from past electric consumption costs of Meralco to determine how its price has changed over the years and what factors cause the price to fluctuate. Svelte was used to build the desktop webpage and Python was utilized to perform various data science techniques on the data. Few graphs were made to show the data in an efficient manner. The results show inconsistent changes in electricity price and how generation and distribution charges heavily affect it.

Alco-OL: Esp32-based Sanitation Dispenser Monitoring System

May 2023 - June 2023

Developed a website that shows current level of content of sanitation dispensers using esp32 as well as their battery status through the internet using Svelte and Firebase. Through this system, users can easily detect which containers need to be refilled. The history log can also be used to detect which dispensers are possibly broken. Though developed for sanitation dispensers, the system can also be used for other purposes (i.e. soap/shampoo dispensers).

Predicting Highest and lowest Region of game Sales using Decision Tree Classifier and other Models (AI)

April 2024 - June 2024

Used Decision Tree Classifier, Multilayer Perception Classifier, and Hist Gradient Boosting Classifier to predict which region has the highest and lowest scale for a video game. Python and Sklearn libraries were used to train different models on a dataset from Kaggle. The results of each classifiers were compared to each other to see their strengths and weaknesses.
